# Traveloop

Traveloop is a smart travel planning web application developed for hackathon evaluation.

The platform helps users:
- Create and manage trips
- Add destinations
- Explore activities
- Generate itineraries
- Estimate travel budgets

---

## Features

- User Authentication
- Trip Creation & Management
- Dynamic Activity Recommendations
- Budget Calculation
- Itinerary Planning
- Responsive UI
- Backend API Integration
- Database Driven Architecture

---

## Tech Stack

### Frontend
- React
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL
